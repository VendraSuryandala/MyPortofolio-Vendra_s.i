Android Development

Dalam project android development ini saya membuat suatu aplikasi pada Android Studio sebagai tugas akhir/skripsi saya. Berikut adalah projectnya:

1. Antarmuka Halaman Login Hasilbumi.id Mobile

Pada Halaman login aplikasi tersebut sudah terhubung kedalam database dengan menggunakan REST API terhadap website hasilbumi.id sehingga saling terintegrasi satu sama lainnya.

2. Halaman Menu Utama

Pada halaman menu utama dengan interface yang memudahkan user dalam melakukan aktivitas dan mengikuti tema yang sesuai dengan website hasilbumi.id.

3. Halaman Kategori produk yang tersedia

Dengan adanya produk tersebut sesuai dengan data yang ada didalam website, sehingga apabila data terubah ataupun terdapat data yang baru maka data produk akan tampil juga di dalam aplikasi hasilbumi.id mobile.

4. Halaman Pemesanan

Apabila user ingin memesan produk tersebut, maka user akan mengisi sebuah data sebagai tempat dikirimnya produk tersebut sesuai dengan alamat yang terinput.
